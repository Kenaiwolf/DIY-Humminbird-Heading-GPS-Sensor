# Contributing

We welcome contributions to improve this DIY heading sensor project!

### How to contribute
- **Add logs:** If you have logs from different Humminbird devices, please upload them to the `/data` folder.
- **Improve Firmware:** Optimize the code for better performance or add support for new compass modules.
- **Documentation:** Help us improve the README or create a wiring guide for other users.

Please ensure all new code includes comments and follows the existing project structure.
